# Milk-Acquiring-System-MAS
**Project Overview :**

  The Milk Acquiring System (MAS) is a digital solution designed to transition manual dairy records into a streamlined, automated platform. As the Lead Planner and Developer , I architected this system to handle dairy registrations, supplier management, and real-time quality reporting.


**Key Features :**

  Dairy Registration: Module for onboarding new dairy entities into the system.
  
  
  Supplier Profiles: Comprehensive management of supplier data and history.
  
  
  Real-time Reporting: Digitization of milk quality reports to replace manual record-keeping.
  
  
  Authentication: Secure user authentication system for data protection.


**Technical Stack :**

  Frontend: HTML5, CSS3, JavaScript.



  Backend: PHP.




  Database: MySQL (Relational Schema Design).




  Tools: XAMPP/WAMP Server, VS Code.

**System Design & Methodology :**

  The development followed the Software Development Life Cycle (SDLC).
  
  
  Data Flow: Designed using Level 0 and Level 1 Data Flow Diagrams (DFDs).
  
  
  Modeling: Utilized Entity-Relationship (ER) models to map database interactions.
  
  
  Testing: Performed Black Box Testing and Database Testing to ensure functional accuracy and data integrity.

**How to Install & Run :**
  Clone this repository.
  
  Import the provided .sql file into your local MySQL database.
  
  
  Configure the database connection in the PHP source files.
  
  Run the project using XAMPP/WAMP.
